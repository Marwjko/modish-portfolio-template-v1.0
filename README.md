## ✨ المميزات

### 🎨 التصميم
- **تصميم عصري واحترافي** - واجهة مستخدم جميلة وحديثة
- **تصميم مستجيب** - يعمل بشكل مثالي على جميع الأجهزة والشاشات
- **وضع ليلي/نهاري** - تبديل سهل بين الأوضاع المختلفة
- **ألوان قابلة للتخصيص** - سهل التعديل والتخصيص

### 🚀 الأداء
- **تحميل سريع** - محسّن للأداء العالية
- **رسوم متحركة سلسة** - تأثيرات بصرية احترافية
- **معايير الويب** - يتبع أفضل الممارسات

### 📱 الأقسام
- **القسم الرئيسي (Hero)** - مقدمة جذابة مع خلفية متحركة
- **قسم عني** - معلومات شخصية ومهارات
- **قسم الأعمال** - عرض المشاريع مع تصفية ديناميكية
- **قسم الخدمات** - عرض الخدمات المقدمة
- **قسم التواصل** - نموذج اتصال وبيانات التواصل
- **التذييل** - روابط سريعة وشبكات اجتماعية

## 📦 التثبيت

### الطريقة 1: التحميل المباشر
1. قم بتحميل الملفات من المجلد
2. فك ضغط الملفات إذا كانت مضغوطة
3. افتح ملف `index.html` في المتصفح

### الطريقة 2: استخدام خادم محلي
```bash
# استخدام Python 3
python -m http.server 8000

# أو استخدام Node.js
npx http-server
```

ثم افتح `http://localhost:8000` في متصفحك.

## 🎯 الاستخدام

### التعديل الأساسي

#### تغيير الألوان الرئيسية
افتح ملف `css/style.css` وعدّل المتغيرات في الجزء العلوي:

```css
:root {
    --primary-color: #6c5ce7;      /* اللون الأساسي */
    --secondary-color: #00b894;    /* اللون الثانوي */
    --accent-color: #ff7675;       /* لون التأكيد */
    /* ... المزيد من الألوان */
}
```

#### تحديث المحتوى
عدّل النصوص والصور في ملف `index.html`:

```html
<!-- تغيير العنوان الرئيسي -->
<h1 class="hero-title">مرحباً، أنا [اسمك]</h1>

<!-- تحديث المعلومات الشخصية -->
<p>أنا متخصص في [تخصصك]</p>
```

#### إضافة أعمال جديدة
أضف عنصر جديد في قسم الأعمال:

```html
<div class="portfolio-item" data-category="web">
    <div class="portfolio-image">
        <div class="image-placeholder">
            <i class="fas fa-globe"></i>
        </div>
        <div class="portfolio-overlay">
            <a href="#" class="portfolio-link">
                <i class="fas fa-external-link-alt"></i>
            </a>
        </div>
    </div>
    <h3>عنوان المشروع</h3>
    <p>وصف المشروع</p>
</div>
```

## 🎨 التخصيص

### تغيير الخطوط
عدّل في `css/style.css`:

```css
body {
    font-family: 'اسم الخط', fallback;
}
```

### تعديل الحجم
استخدم المتغيرات المعرّفة أو عدّل القيم مباشرة:

```css
.hero-title {
    font-size: 60px;  /* غيّر هنا */
}
```

### إضافة صور مخصصة
استبدل `image-placeholder` بصورة حقيقية:

```html
<img src="assets/images/your-image.jpg" alt="وصف الصورة">
```
