# 🚀 البدء السريع - نشر الموقع في 5 دقائق

اتبع هذه الخطوات البسيطة لنشر موقعك على الإنترنت مجاناً!

---

## ⚡ الخيار 1: Netlify (الأسهل) ⭐

### الخطوة 1: فك الضغط
```
فك ضغط ملف modish-portfolio-template-complete.zip
```

### الخطوة 2: إنشاء حساب GitHub
1. اذهب إلى https://github.com
2. انقر "Sign up"
3. أكمل التسجيل

### الخطوة 3: رفع الملفات إلى GitHub
1. انقر "New" → "New repository"
2. اسم المستودع: `modish-portfolio`
3. اختر "Public"
4. انقر "Create repository"
5. اتبع التعليمات لرفع الملفات

### الخطوة 4: نشر على Netlify
1. اذهب إلى https://netlify.com
2. انقر "Sign up"
3. اختر "GitHub"
4. اختر مستودعك
5. انقر "Deploy"

✅ **تم! موقعك الآن على الإنترنت!**

---

## ⚡ الخيار 2: Vercel

### الخطوة 1: إنشاء حساب
1. اذهب إلى https://vercel.com
2. انقر "Sign Up"
3. اختر "GitHub"

### الخطوة 2: استيراد المشروع
1. انقر "Import Project"
2. الصق رابط مستودع GitHub
3. انقر "Import"

### الخطوة 3: النشر
1. اترك الإعدادات الافتراضية
2. انقر "Deploy"

✅ **تم! موقعك الآن على الإنترنت!**

---

## ⚡ الخيار 3: GitHub Pages (الأسرع)

### الخطوة 1: إنشاء مستودع
1. اذهب إلى https://github.com
2. انقر "New repository"
3. اسم المستودع: `username.github.io`
4. اختر "Public"

### الخطوة 2: رفع الملفات
```bash
cd modish-portfolio-template
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/username/username.github.io.git
git push -u origin main
```

### الخطوة 3: انتظر دقيقة
موقعك سيكون متاحاً على: `https://username.github.io`

✅ **تم! موقعك الآن على الإنترنت!**

## 📝 تخصيص الموقع

### 1. تغيير الاسم والعنوان
افتح `index.html` وعدّل:
```html
<h1 class="hero-title">مرحباً، أنا [اسمك]</h1>
```

### 2. تغيير البريد والهاتف
ابحث عن "تواصل معي" وعدّل:
```html
<p>contact@modish.com</p>
<p>+966 50 123 4567</p>
```

### 3. تغيير الألوان
افتح `css/style.css` وعدّل:
```css
--primary-color: #6c5ce7;  /* اللون الأساسي */
```

### 4. إضافة صورك
ضع صورتك في `assets/images/` واستبدل:
```html
<img src="assets/images/your-image.jpg" alt="صورتي">
```

## 🔗 إضافة نطاق مخصص (اختياري)

إذا أردت نطاقاً مثل `myname.com`:

1. **اشتر نطاق** من:
   - Namecheap.com
   - GoDaddy.com
   - Domain.com

2. **ربط النطاق**:
   - في Netlify: Settings → Domain Management
   - في Vercel: Settings → Domains
   - اتبع التعليمات
