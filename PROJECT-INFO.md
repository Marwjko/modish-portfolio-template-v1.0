- تأكد من أن جميع الملفات موجودة

## 📁 البنية الأساسية

### شرح الملفات

```
modish-portfolio-template/
│
├── index.html                 # الملف الرئيسي - يحتوي على كل محتوى الموقع
│
├── css/
│   └── style.css             # جميع الأنماط والتصاميم
│
├── js/
│   └── script.js             # الوظائف التفاعلية والديناميكية
│
├── assets/
│   └── images/               # مجلد الصور والوسائط
│
├── README.md                 # ملف التعريف الأساسي
├── LICENSE.md                # ملف الترخيص وحقوق الملكية
└── DOCUMENTATION.md          # هذا الملف
```

### شرح الملفات الرئيسية

#### 1. index.html
يحتوي على:
- هيكل الصفحة الكامل
- جميع الأقسام والمحتوى
- الروابط إلى ملفات CSS و JavaScript

#### 2. css/style.css
يحتوي على:
- المتغيرات الأساسية (الألوان والخطوط)
- أنماط جميع العناصر
- الرسوم المتحركة
- استجابة التصميم (Responsive)

#### 3. js/script.js
يحتوي على:
- تبديل الوضع الليلي/النهاري
- قائمة الموبايل
- تصفية الأعمال
- معالجة النموذج

## 🎨 التخصيص

### تغيير الألوان الأساسية

افتح `css/style.css` وابحث عن:

```css
:root {
    --primary-color: #6c5ce7;      /* اللون الأساسي */
    --secondary-color: #00b894;    /* اللون الثانوي */
    --accent-color: #ff7675;       /* لون التأكيد */
    --dark-bg: #1a1a2e;            /* خلفية الوضع الليلي */
    --light-bg: #f8f9fa;           /* خلفية الوضع النهاري */
    --text-dark: #2d3436;          /* لون النص الداكن */
    --text-light: #636e72;         /* لون النص الفاتح */
}
```

**مثال تغيير اللون الأساسي:**
```css
--primary-color: #3498db;  /* تغيير من البنفسجي إلى الأزرق */
```

### تغيير النصوص

#### تغيير العنوان الرئيسي (Hero Title)
في `index.html`، ابحث عن:
```html
<h1 class="hero-title">مرحباً، أنا مصمم ومطور</h1>
```
غيّره إلى:
```html
<h1 class="hero-title">مرحباً، أنا [اسمك]</h1>
```

#### تحديث النبذة الشخصية
ابحث عن قسم "عني" وعدّل النصوص:
```html
<p>أنا متخصص في تطوير الويب وتصميم الواجهات الحديثة.</p>
```

#### تحديث معلومات التواصل
في قسم التواصل، عدّل:
```html
<p>contact@modish.com</p>
<p>+966 50 123 4567</p>
<p>الرياض، المملكة العربية السعودية</p>
```

### إضافة صور مخصصة

#### استبدال الصور الموجودة
ابحث عن `image-placeholder` واستبدلها:

```html
<!-- قبل -->
<div class="image-placeholder">
    <i class="fas fa-user-circle"></i>
</div>

<!-- بعد -->
<img src="assets/images/profile.jpg" alt="صورتي الشخصية">
```

#### إضافة صورة للأعمال
```html
<div class="portfolio-image">
    <img src="assets/images/project-1.jpg" alt="وصف المشروع">
    <div class="portfolio-overlay">
        <a href="https://example.com" class="portfolio-link">
            <i class="fas fa-external-link-alt"></i>
        </a>
    </div>
</div>
```

### تعديل الخطوط

في `css/style.css`:
```css
body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}
```

**خطوط عربية موصى بها:**
```css
font-family: 'Cairo', 'Droid Arabic Kufi', sans-serif;
```

## 🔧 الميزات المتقدمة

### 1. الوضع الليلي/النهاري

يتم التحكم به تلقائياً عبر JavaScript:

```javascript
// تفعيل الوضع الليلي
document.body.classList.add('dark-mode');

// تعطيل الوضع الليلي
document.body.classList.remove('dark-mode');
```

يتم حفظ التفضيل في `localStorage`.

### 2. تصفية الأعمال

يمكن إضافة فئات جديدة:

```html
<!-- إضافة زر تصفية جديد -->
<button class="filter-btn" data-filter="branding">هوية بصرية</button>

<!-- إضافة عنصر عمل جديد -->
<div class="portfolio-item" data-category="branding">
    <!-- المحتوى -->
</div>
```

### 3. نموذج الاتصال

يمكن ربطه بخدمة إرسال بريد:

```javascript
// في js/script.js
contactForm.addEventListener('submit', (e) => {
    e.preventDefault();
    
    // أرسل البيانات إلى خادم
    fetch('/send-email', {
        method: 'POST',
        body: new FormData(contactForm)
    });
});
```

### 4. الرسوم المتحركة

تتم عبر CSS و JavaScript:

```css
@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```

### 5. الملاحظة التفاعلية (Intersection Observer)

يتم استخدامها لتشغيل الرسوم المتحركة عند الوصول:

```javascript
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.style.animation = 'fadeInUp 0.6s ease forwards';
        }
    });
});
```

## 📱 الاستجابة (Responsive Design)

القالب يستجيب تلقائياً لجميع أحجام الشاشات:

- **سطح المكتب:** 1200px وأكثر
- **الأجهزة اللوحية:** 768px - 1199px
- **الهاتف الذكي:** أقل من 768px

### تعديل نقاط التوقف

في `css/style.css`:

```css
@media (max-width: 768px) {
    /* أنماط الأجهزة اللوحية والهاتف */
}

@media (max-width: 480px) {
    /* أنماط الهاتف الذكي فقط */
}
```

## 🐛 استكشاف الأخطاء

### المشكلة: الموقع لا يظهر بشكل صحيح

**الحل:**
1. تأكد من أن جميع الملفات في نفس المجلد
2. تحقق من أسماء الملفات (حساسة لحالة الأحرف)
3. امسح ذاكرة المتصفح

### المشكلة: الصور لا تظهر

**الحل:**
1. تحقق من مسار الصورة
2. تأكد من وجود الصورة في مجلد `assets/images`
3. استخدم المسار الصحيح: `assets/images/image-name.jpg`

### المشكلة: الرسوم المتحركة لا تعمل

**الحل:**
1. تأكد من أن JavaScript مفعّل
2. افتح وحدة تحكم المتصفح (F12) وتحقق من الأخطاء
3. تأكد من تحميل ملف `script.js`

### المشكلة: الألوان غير صحيحة

**الحل:**
1. امسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
2. أعد تحميل الصفحة (Ctrl+F5)
3. تحقق من قيم الألوان في `style.css`

### المشكلة: الموقع بطيء

**الحل:**
1. ضغط الصور
2. استخدام خادم CDN للملفات الثقيلة
3. تقليل عدد الرسوم المتحركة
